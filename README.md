# DLE Skid 3D

Интерактивная 3D-визуализация скида установки прямого извлечения лития (Direct Lithium Extraction, DLE).
Полностью самодостаточная страница: библиотека three.js встроена в `index.html`, внешние зависимости и сеть не нужны.

## Демо

**https://igorkant.github.io/dle-skid-3d/**

## Использование

- **Онлайн** — откройте ссылку выше (GitHub Pages).
- **Локально** — скачайте `index.html` и откройте его в браузере двойным щелчком. Интернет не требуется.

## Управление сценой

- **ЛКМ + перетаскивание** — вращение камеры вокруг скида
- **Колесо мыши** — приближение / отдаление
- **ПКМ + перетаскивание** — панорамирование

## Технологии

- [three.js](https://threejs.org/) — рендеринг WebGL (встроена в файл)
- Один HTML-файл, без сборки и без внешних запросов

## Обновление

```bash
cp /путь/к/новой_версии.html index.html
git add index.html
git commit -m "Update DLE skid 3D scene"
git push
```

После `push` в ветку `main` GitHub Pages пересобирается автоматически.
